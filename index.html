调研背景

用数据与洞察建立信任感 · 深度 1v1 · 20–30 岁都市女性 · 一线在校生 / 白领

访谈维度覆盖：EMS 认知、购买决策、审美偏好、生活方式关联。以下六条核心洞察均直接对应一条设计决策。

洞察 01

「科技感 = 安全可靠」，而非炫技

「你走进一个美容院，看到他们那么有科技感，就觉得它肯定是安全的——这是最重要的……」
→

设计决策：科技感服务于信任。视觉做「精密医仪感」——冷静、可靠，不做科幻炫技。

洞察 02

「医疗白 + 警示红」= 恐惧

「假如它做得像医疗设备，那种白色然后中间几个按钮是红色的……你可能会看到就觉得怕了。」
→

设计决策：禁用冷白 + 警示红。主色改为玫瑰裸 #E2CACC —— 保留专业感，去除恐惧感。

洞察 03

莫兰迪 = 「选它不会输错」

「莫兰迪这种……大部分人都会喜欢，有种『选择它不会输错的感觉』……」
→

设计决策：全链路低饱和莫兰迪调：主色 #E2CACC、辅色 #96AFC8、页面背景 #F5F0EB。

洞察 04

圆润感 = 想触摸

「C 第一眼吸引我的是它突起来的圆润线条……那个按钮让我一看到就很想去按它。」
→

设计决策：产品正面倒角 R96、侧面 R32；UI 按钮全圆角胶囊，卡片圆角 12px。

洞察 05

跳色带来「玩具感」

「如果没有那个跳色（橙色），会不会看起来稍微……可信度高一点？」
→

设计决策：禁止高饱和跳色；辅色严控为烟蓝 / 薰衣草灰；材质哑光磨砂提升专业感。

洞察 06

小红书 = 女性生活方式分享

「主要是小红书，感觉更集中于女生之间分享生活经验的一个软件。」
→

设计决策：叙事改为「有仪式感的生活方式」—— 场景化、真实皮肤、暖光，避免功能硬广罗列。

推导式流程（工业设计逻辑）

  1. 市场分析（差异化点）
  2. 用户研究（产品特性 / 用户画像 / 使用场景）
  3. 设计基因关键词（极简 × 圆润 × 莫兰迪 × 专业可信赖）
  4. 视觉元素提取（参考品牌 → 迁移至 VONMIE）
  5. 设计规范落地（颜色 / 字体 / 圆角 / 组件 / 材质）

品牌基因分析

展示如何从标杆品牌学习，但不抄袭；踩雷表直接映射 VONMIE 对策（与需求文档 PART 2 一致）

2.1 标杆品牌一：Lululemon

✅ 好在哪里

① 锁定精准人群——Super Girls 创始人 Chip Wilson 在 1998 年发现北美女性跑者占 60%，她们讨厌「被打量、被灌输性感」，愿意为「取悦自己的体验」付费。Lululemon 不是在卖瑜伽服，是在卖一种「我懂你」的品牌认同感。

② 极致的材质技术体系（不是贴标，是真研发） 九大核心面料技术，每种都有明确的疏水/吸湿功能。Nulux 解决了瑜伽裸感和高支撑的矛盾——技术有疆界，不是空话。

③ 稀缺 + 精品店策略 = 高客单价 + 社区感 不走大众流通，店不多但都在黄金地段。卖的是饮食、锻炼、陪伴、归属感——不是卖货，是卖生活方式。

④ 社群运营是真护城河 Store Ambassador 体系 + 社区活动，让用户主动传播。粉丝不是买来的，是培养出来的。

⑤ 从运动延伸到精致日常，顺序对了 先建立专业可信度，再扩展生活场景。

逻辑链：专业可信度 → 扩展生活场景

❌ 踩的雷（VONMIE 要避开）

雷区 Lululemon 发生了什么 VONMIE 的对策
品牌空心化 董事会让撞款，最懂设计的 CEO 离开，产品 GAP 化 规模是毒药，KPI 不能压倒创意，设计决策权要保护
依赖单一市场 美洲本土增速只剩 3%，过度依赖中国增量 从第一天建立清晰品牌资产，不依赖单一渠道
扩品类稀释 男装/配件让「这不是为我做的」 前期死守 24 岁核心人群，先锐利后扩展

颜色决策影响

Lululemon 的启示：用「体育科技黑」建立专业感，用「季节马卡龙色」制造话题——两个颜色承载两种叙事。

VONMIE 的转译：#2D3A45（深不锈钢）= 专业底色；#E2CACC（玫瑰裸）= 温柔亮色。两者构成品牌视觉的核心张力。

2.2 标杆品牌二：Stanley

✅ 好在哪里

① 精准转轨：百年户外硬汉 → 时髦女性市场 2019 年通过 KOL 合作，将一款功能保温杯变成 TikTok 社交货币。叙事重构让「不太可能」的消费品成为抢手货。

② 饥饿营销 + 限量色系 = 情绪消费 马卡龙色、联名款、限购 → 利用 FOMO（错失恐惧症）驱动购买。消费者买的不是杯子，是「我也有」的归属感。

③ 产品质量托底 百年积累的硬核品质背书，让网红红利退去后不至于直接垮掉。

❌ 踩的雷（VONMIE 要避开）

雷区 Stanley 发生了什么 VONMIE 的对策
品牌割裂 硬汉工具品牌急转少女粉色，原有用户被抛弃 从第一天定义清楚「专业的温柔」，不左右横跳
营销疲劳 过度限量炒作，被批评「玩套路」 联名要谨慎，视觉系统要有强势主张
质量召回 2024 年 260 万产品召回，顶流瞬间跌落 产品安全 > 一切，材质规范从设计层锁死
热度是流行周期 《纽约客》：TikTok 驱动红利本质是流行周期 品牌资产不能建立在热度上，要建立在品质上

颜色决策影响

Stanley 的启示：颜色是叙事工具，不是装饰。「田园绿」让 Stanley 从工地来到野餐布——一个颜色完成品牌形象跃迁。

VONMIE 的转译：#E2CACC（玫瑰裸色）是我们的叙事颜色——它让 EMS 仪器从「医疗设备」变成「梳妆台上的精致单品」，完成同样的形象跃迁。

「下一个卖爆的一定是哑光烟粉色——这个颜色的女生需要的松弛感，是能共情的情绪点。」

2.3 核心结论（一句话版）

Lululemon 告诉你：找到精准的人，建立真实的技术体感，用统一的设计语言说话。

Stanley 告诉你：重构叙事，但产品是底线。

VONMIE 要做的：用 Lululemon 的精准 + Stanley 的叙事爆发力，但避开两者各自踩的那些雷。

设计禁忌清单

PART 6 · 执行红线

  • 禁止使用纯白 #FFFFFF 作为页面背景（使用 #F5F0EB）
  • 禁止使用纯黑 #000000 作为正文色（使用 #2D3A45）
  • 禁止冷白 + 警示红配色（用户明确表达恐惧感）
  • 禁止高饱和跳色（饱和度 > 60%）
  • 禁止方形直角容器（最小圆角 4px）
  • 禁止标题字重超过 500
  • 禁止镜面 / 亮面质感的 UI 元素
  • 禁止「功能罗列式」视觉叙事（须场景化、仪式感）
